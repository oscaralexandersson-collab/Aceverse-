
# Aceverse Development Log

## [1.2.0] - Total Database Rewrite & AI Voice Integration
- **Schema Overhaul**: Migrated entire database to a normalized snake_case PostgreSQL schema in Supabase.
- **RLS Lockdown**: Implemented comprehensive Row Level Security policies for GDPR Article 32 compliance.
- **Voice Mode**: Added low-latency AI verbal coaching using Gemini 2.5 Flash Live API.
- **Unified Sync**: Standardized naming across frontend components and backend services.
- **Marketing Engine**: Added support for Brand DNA extraction and multi-channel AI campaigns.
- **Auto-Provisioning**: Profiles are now automatically created via database triggers upon user signup.
